Hawaii Framework is a Java framework for developing Spring based applications.

Combining Spring Boot and the Hawaii Framework production-ready features and auto configuration brings even more power and simplicity - without sacrificing flexibility - to developers.

Foundation

  • Java 8
  • Spring Framework
  • Spring Boot

Features

  • Hawaii opinionated auto configuration
  • Hawaii Starters - based on Spring Boot architecture
  • Hawaii Logging - advanced E2E transaction and call logging
  • Hawaii Validation (including detailed REST errors) - inspired on Spring's Validator/Errors
  • Hawaii REST sugar
  • Hawaii Test sugar
  • Much more to come...
